# OpenCalc
A PicoCalc-based command line calculator firmware

## Functions
| Command | Description | Usage | Implemented |
| --- | --- | --- | --- |
| ver | Prints the OpenCalc version number | ver | Y |
| cls | Clears the screen | cls | Y |
| bat | Prints the battery percentage | bat | N |
| +, -, *, /, () | Basic arithmetic | 3(4-5)/7+25*5 | Y |
| ^, !, sqrt, cbrt | Advanced arithmetic | 2^3, 32!, sqrt(9) | Y |
| pi, e, i | Mathematical constants: pi, Euler's number, imaginary unit | 2pi | Y |
| sin, cos, tan, asin, acos, atan, sinh, cosh, tanh | Trigonometry | cos(45) | Y |
| x, ans | Variables | 2x^2+4x+8, ans/45 | Y |
| log, ln, exp, abs, floor, ceil, round, sign | Other |  | Y |Other |  | Y |Other |  | Y ||  |  |  |  |os(45) | Y |os(45) | Y |os(45) | Y |rigo |  |  |  |  |  |  |  |  |  |  |  |  |  |  | |  |  |  |t(9) | Y |sqrt(9) | Y |t(9) | Y |c |  | Y |  |  | Y ||  |  |  |ns
