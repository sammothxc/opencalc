# OpenCalc
A PicoCalc-based command line calculator firmware

## Functions
| Command | Description | Usage |
| --- | --- | --- |
| ver | Prints the OpenCalc version number | ver |
| cls | Clears the screen | cls |
| bat | Prints the battery percentage | bat |
| +, -, *, /, () | Basic arithmetic | 3(4-5)/7+25*5 |
| ^, !, sqrt, cbrt | Advanced arithmetic | 2^3, 32!, sqrt(9) |
| pi, e, i | Mathematical constants: pi, Euler's number, imaginary unit | 2pi |
| sin, cos, tan, asin, acos, atan, sinh, cosh, tanh | Trigonometry | cos(45) |
| all non-mathematical constant letters, ans | Variables | 2a^2+4b+8^c, ans/45 |
| clenv | Clear all variables | clenv |
| log, ln, exp, abs, floor, ceil, round, sign | Other | TBD |
